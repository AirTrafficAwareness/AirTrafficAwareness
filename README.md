# AirTrafficAwareness

Welcome, The AirTrafficAwareness Project is broken up into Client and Server.

To run Client from root directory on CLI run:

`cd Client`

`npm ci`

`npm install`

To run Server from root directory on CLI run:

`cd Server`

`npm ci`

`npm install`
